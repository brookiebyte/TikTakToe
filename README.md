# TikTakToe
TikTakToe is BrookieByte's git practice version of implementing TicTacToe as a terminal game

## To Run
### Linux / MacOS
Run in terminal
```
python3 play_tik_tak_toe.py
```
### Windows
Run in Terminal
```
python play_tik_toe.py
```
